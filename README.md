# RockPaperScissors

<p align='center'><img src="/RPS_demo.gif" width="75%" height="75%"/></p>

The classic game of rock, paper, scissors; now with visuals!

## Installation

That's the neat part, no external libraries are required!

## How to Run

Play the game by running:

```sh
python3 rock_paper_scissors.py
```

For those who prefer to play without any visuals, try the old school mode:

```sh
python3 rock_paper_scissors_old_school.py
```
